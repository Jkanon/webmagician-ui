<h1 align="center">Webmagician UI</h1>

<div align="center">

ウェブ・マジシャン(Webmagician)の UI、ant-design-proで作られている。

## 使用法

### bash を使う方法

```bash
$ npm install
$ npm start         # http://localhost:8000 を開く
```
